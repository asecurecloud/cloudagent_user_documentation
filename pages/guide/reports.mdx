## Running a New Report

1. Navigate to the **Reports** page or workload details.
2. Click **+ Run New Report**.
3. Select the report type from the available options.
4. Choose the target environment(s).
5. Click **Generate** to start the report.

<Callout type="info" emoji="â±ï¸">
  **Processing Time**: Reports typically complete within 5-15 minutes depending on the size of your environment.
</Callout>

## Viewing Report Details

Click **View Report** on any completed report to see:

- **Executive Summary**: High-level overview of findings
- **Detailed Findings**: Individual check results with pass/fail status
- **Affected Resources**: Specific resources that need attention
- **Remediation Steps**: Recommended actions to resolve issues
- **Export Options**: Download as PDF for sharing with auditors

![Report Detail View](/images/report_detail.png)

## Scheduling Reports

For continuous compliance monitoring:

1. Click on a report type.
2. Select **Schedule**.
3. Choose frequency (Daily, Weekly, Monthly).
4. Set notification preferences.
